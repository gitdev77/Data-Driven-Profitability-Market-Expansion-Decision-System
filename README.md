# Data-Driven Profitability & Market Expansion Dashboard (Tableau)

## Overview

This project builds a structured SQL analytics pipeline and an interactive Tableau dashboard to support profitability optimization and market expansion decisions.

The dashboard answers key business questions:
- Which regions drive maximum revenue?
- Where are margins highest?
- Which categories underperform?
- Where should expansion efforts be prioritized?

---

## Tools Used

- SQL (Data Cleaning & KPI Aggregations)
- Tableau Desktop (Dashboard & Visualization)
- CSV Dataset (Sales Data)

---

## Analytical Framework

### 1. Revenue Share by State
Identifies high-contributing markets.

### 2. Efficiency Analysis
Evaluates relationship between order volume and revenue.

### 3. Fulfillment Performance
Tracks operational efficiency across regions.

### 4. Category Performance
Highlights product-level profitability trends.

## Key Insights

- Revenue concentration observed in top-performing states.
- Certain mid-volume regions demonstrate higher margins.
- Fulfillment efficiency impacts overall profitability.
- Expansion strategy should focus on high-margin, under-penetrated markets.

Transforms raw sales data into a decision-support system for strategic growth planning.
